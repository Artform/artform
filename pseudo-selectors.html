Pseudo-Selectors are used for selecting objects according to a particular case. More examples of these (which are not covered here) are :hover, :focus, :last-of-type, and many more.

  • Toggle all boxes using nth-child
    • .box:nth-child(odd) - odd boxes
    • .box:nth-child(even) - even boxes
    • .box:nth-child(3n) - every third box
    • .special:nth-child(odd) - odd boxes with special class
    • .special:nth-child(even) - even boxes with special class
  • More complex examples:
    • ul ul li:first-child - first list item in each unordered sublist
    • #boxes .box:last-child - last box
